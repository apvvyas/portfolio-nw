## How to Use

1. **Add Your Profile Photo** (Important!)
   - Go to https://www.facebook.com/vyasapoorv
   - Right-click on your profile photo and select "Save Image As..."
   - Save it as `profile-photo.jpg` in this folder
   - Or use any photo file and update the filename in `index.html` (line 47)

2. Open `index.html` in a web browser
3. Navigate through sections using the navigation menu
4. Update content as needed:
   - Personal information in `index.html`
   - Styling and colors in `styles.css`
   - Interactive features in `script.js`

## Customization

### Update Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #1e40af;
    --accent-color: #3b82f6;
    /* ... */
}
```

### Update Content
Edit the HTML content in `index.html` to personalize:
- Name and title
- About section text
- Experience timeline
- Skills list
- Contact information

### Add More Sections
Follow the existing section structure in `index.html` and add corresponding styles in `styles.css`.
